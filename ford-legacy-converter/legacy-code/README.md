# Legacy C Code Example

This folder contains a simple C program example that can be used to test the legacy code converter.

## Files

- `main.c` - Main calculator program with basic arithmetic operations
- `utils.h` - Header file with utility function declarations
- `utils.c` - Implementation of utility functions

## Building

To compile the C program:

```bash
gcc -o calculator main.c utils.c
./calculator
```

## Features

- Basic arithmetic operations (add, subtract, multiply, divide)
- Error handling for division by zero
- Structure-based result handling
- Input/output operations
- Header file organization

This code can be converted to modern languages like TypeScript, Python, or Java using the legacy code converter.
